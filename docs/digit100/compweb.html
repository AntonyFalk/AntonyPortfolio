During my time in my Intro to Digital Humanities class, I learned how to make a website using two different programs, Github Pages and Wordpress. I will be comparing these programs and the experiences I've had using them.

What made using Github Pages particularly difficult was the fact that it was completely new to me, and I had to learn all of the commands from scratch. There was definitely a steep learning curve when I was working with this software. Wordpress was easier in this sense because there was a Graphic User Interface that allowed me to do whatever I wanted with the website without any previous coding knowledge.
